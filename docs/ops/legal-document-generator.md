# 法务材料生成器

本模块用于在动作中心快速生成法务材料初稿，并将结果沉淀到 `legal_document` 执行工单。

## 支持材料

- 民事起诉状：适用于常规诉讼立案材料初稿。
- 财产保全申请书：适用于查封、扣押、冻结车辆或等值财产的材料初稿。
- 实现担保物权特别程序申请书：适用于车辆已收回、已入库且满足内部准入条件后的特别程序材料初稿。

## API

- `POST /api/legal-documents/generate`
- 需要登录，并至少具备 `operator` 角色。
- 可选传入 `work_order_id`。如果传入，必须关联当前租户下的 `legal_document` 工单，生成成功后该工单会被置为 `completed`，并在 `result` 中记录材料类型、标题、生成时间和 HTML。

## 风控边界

- 模板使用 Jinja autoescape，避免用户输入直接注入 HTML。
- 生成内容只作为业务流转和法务复核初稿，不作为最终法律意见。
- 特别程序材料模板会提示操作人员复核车辆收回、入库及逾期准入条件。
